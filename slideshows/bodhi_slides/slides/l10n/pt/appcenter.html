• O AppCenter do Bodhi Linux é uma forma simplificada de encontrar excelentes aplicações para o Bodhi.

  • Também fornecemos o Synaptic e o apt para instalar uma grande quantidade de software dos repositórios do Ubuntu e dos nossos próprios repositórios.
